## 5.1 필터링

### 5.1.1 프레디케이트로 필터링
- 스트림 인터페이스는 filter 메서드를 지원한다.
  - filter 메서드는 **프레디케이트** (불리언을 반환하는 함수)를 인수로 받아서 프레디케이트와 일치하는 모든 요소를 포함하는 
스트림을 반환한다.

### 5.1.2 고유 요소 필터링
- 스트림은 고유 요소로 이루어진 스트림을 반환하는 distinct 메서드도 지원한다
  - (고유 여부는 스트림에서 만든 객체의 hashCode, equals로 결정된다.)
    - 중복을 제거함
