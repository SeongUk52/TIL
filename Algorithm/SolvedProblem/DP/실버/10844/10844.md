# 쉬운 계단 수
### 실버 1 DP
[문제](https://www.acmicpc.net/problem/10844)

[파이썬 코드](10844.py)

---

- 앞에 오는 숫자 = 0 
  - dp[자리 수][0] = dp[자리 수 - 1][1]


- 앞에 오는 숫자 = 1~8
  - dp[자리 수][앞에 오는 숫자] = dp[자리 수 - 1][앞에 오는 숫자 -1] + dp[자리 수 - 1][앞에 오는 숫자 + 1]
  - dp[n][i] = dp[n][i-1] + dp


- 앞에 오는 숫자 = 9 
  - dp[자리 수][9] = dp[자리 수 - 1][8]
